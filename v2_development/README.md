# v2_development - New Version with Auth

This is the development version with:
- Descope authentication (Google login)
- User quotas
- Stripe payments

## Setup:
```bash
cd v2_development
pip install -r requirements.txt
```

## Run locally:
```bash
python app.py --port 5001
# Opens on http://localhost:5001
```
